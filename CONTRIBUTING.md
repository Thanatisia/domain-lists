# CONTRIBUTING

Please review the following steps to contribute to the adlists

## Contribution Steps
> The following are basically the typical Open Source Software Development contribution process using git (aka 'GitOps')
- Clone/pull the latest changes of the repository
    + Please do this before you start making any changes
    - Cloning
        ```console
        git clone [repository-url]
        ```
    - Pull
        ```console
        git pull
        ```

## Basic Rules
1. Please create a Pull Request when submitting changes for review and merger
2. Ensure that the Pull Request follows the following template for consistency
    ```
    List Type: {Blacklist/Whitelist]
    List Name: 
    Number of Domains Added: 
    Number of Domains Removed: 
    Changes Made:
        - White List
            + [Changes Here]
        - Black List
            + [Changes Here]
    ```
